# General Settings

in the your `nvoidrc.lua` add the following:

```lua
M.options = {
  clipboard = 'unnamedplus',
  cmdheight = 1,
  mouse = 'a',
  mapleader = ' ',
  wrap = false,
  number = true,
  relative_number = false,
  number_width = 6,
  cursor_line = true,
  hidden = true,
  expand_tab = true,
  ignore_case = true,
  smart_case = true,
  smart_indent = true,
  swap_file = false,
  backup = false,
  show_mode = false,
}
```

start editing :D
