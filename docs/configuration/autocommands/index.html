Commands

Autocmds​

you can autocommands with:

nvoid.autocommands = {
    {
        "autocmd events",
        { 
            pattern = { "pattern" },
            command = "vimscpript",
        }
    },
}

You can also add lua callbacks

nvoid.autocommands = {
    {
      "autocmd events", {
      pattern = {"pattern" },
      callback = function()
        -- lua function
      end
    },
  }
}
